# Noida Falcons Website

A modern responsive MVP homepage for **Noida Falcons** — a premium cricket franchise brand with dark sports styling, glassmorphism, Framer Motion animations, animated counters, countdown UI, squad cards, news cards, sponsors carousel, and mobile navigation.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Brand

- Team: Noida Falcons
- Tagline: Rise Above The Rest
- Jersey Orange: `#FF6A00`
- Sky Blue Accent: `#0A8FD8`
- Midnight Black: `#0A0A0A`
- Silver: `#C0C0C0`
- White: `#FFFFFF`

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Quality Checks

```bash
npm run lint
npm run build
```

Both checks have been run successfully.

## Main Sections

- Sticky glassmorphism navigation with mobile hamburger menu
- Full-screen animated cricket stadium hero
- About / vision / mission section
- Featured player cards
- Upcoming match card with live countdown
- Animated team statistics counters
- Latest news cards
- Auto-scrolling sponsor carousel
- Footer with quick links and social platforms

## Notes

The MVP intentionally uses CSS/SVG-driven visuals and reusable components, so it looks polished without requiring a backend, CMS, or external image assets.
