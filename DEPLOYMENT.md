# Deploy Noida Falcons to Vercel

The project is ready for Vercel deployment using Vercel's native Next.js builder.

## Option 1: Deploy from GitHub — Recommended

1. Create a new GitHub repository.
2. Push this project to that repository.
3. Go to [https://vercel.com/new](https://vercel.com/new).
4. Import the GitHub repository.
5. Vercel should auto-detect the project, but confirm these settings:

   - Framework Preset: `Next.js`
   - Install Command: `npm ci`
   - Build Command: `npm run build`
   - Output Directory: leave blank / default

6. Click **Deploy**.
7. Vercel will give you a public URL like:

   ```text
   https://noida-falcons.vercel.app
   ```

## Option 2: Deploy with Vercel CLI

From the project folder, run:

```bash
npm install
npm run build
npx vercel login
npx vercel --prod
```

Follow the prompts. When deployment finishes, Vercel will print your live URL.

## Local Development

```bash

npm run dev
```

Open:

```text
http://localhost:3000
```

